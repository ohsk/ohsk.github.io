Mobility Tech

Curbs API

Curbs are the nexus of mobility in cities, but information about the rules that govern them is difficult to obtain. Coord's Curbs API allows developers to easily understand how any stretch of curb can be used for over 200K curbs in Los Angeles, New York City, San Francisco, and Seattle.
